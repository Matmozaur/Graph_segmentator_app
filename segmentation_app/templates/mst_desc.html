Minimal Spinning Tree

Parameters:

Type of graph

-8 edges
-4 edges

Threshold function

-edge weight + const * 1.0 * sqrt(size of a segment)
-edge weight + const * 1.0 / sqrt(size of a segment)
-const

Const

Real number

Minimal segment size

Integer


Steps:

1. Prepare graph

a) Create set of width*height (of image) vertices where each vertex corresponding to pixel on image.
b) Depending on type of graph argument connect each vertex witch its upper, down left and right neighbors, and maybe oblique neighbors.
c) Set weights of edges as eucklidean distances between RGB vectors corresponding to vertices.

2. Run MST with border function

a) Initialise forest where each tree is one vertex from base graph.
b) Sort all edges ascending.
c) For each edge in that order if adding it wont cause a cycle and its value is less or equal than given threshold function (defined by parameters threshold functions and const) add edge to forest.
d) Return forest as segmentation where each segment is a different tree in it.

3. Reduce noise

a) Sort all edges ascending.
b) For each edge in that order if adding it wont cause a cycle and at least one of connected by it segments is smaller then parameter minimal segment size add this edge to a forest.
