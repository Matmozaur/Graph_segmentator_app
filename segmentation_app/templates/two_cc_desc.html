Two-connected components

Parameters:

  • Channel:
    • All
    • Red (recommended for bees)
    • Green
    • Blue (recommended for blood cells)
  • Threshold:
    • Otsu's threshold
    • Const
  • Const (if was choosen above):
    • Real number
  • Filling in holes:
    • Yes/No

Steps:

  1. Prepare image
    1. Resize image if it is too big.
    2. Enhance image (contrast, brightness, sharpness).
  2. Convert image to binary version:
    1. Take choosen channel(s) under consideration.
    2. If user chose all channels, convert image into grayscale.
    3. Get binary version of image with use proper threshold (Otsu's threshold or const number).
    4. Fill in holes (if proper option was choosen).
  3. Prepare graph
    1. Create set of width*height (of image) vertices where each vertex corresponds to pixel on image.
    2. Add edges to the graph: connect each vertex witch its upper, down, left and right neighbors.
  4. Get two-connected components of graph
    1. Using proper algorith, get k-connected components of graph and choose k=2.
    2. Reduce too small and too big components (eg. background).
    3. Visualize effects of segmentation by assigning random color to each component.
    4. Return segmented image and number of components.
