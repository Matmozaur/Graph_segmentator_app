Interactive Segmentation

Parameters:

Foreground sample

This is a subimage of the original image which belongs to its foreground. Will be used to find a probability distribution describing image regions belonging to the foreground.

Background sample

This is a subimage of the original image which belongs to its background. Will be used to find a probability distribution describing image regions belonging to the background.

Steps:

1. Construction of directed graph that represents our image.

Every pixel is a node in this graph. Nodes are connected if they share an edge in the image. We add the 'source' and 'sink' nodes that are connected with every node in this graph.

2. Assigning weights to edges of the constructed graph.

Based on probabilities describing if particular pixel belongs more to the foreground or background of the image.

3. Solving a max-flow/min-cut problem on the

There are several standard algorithms, e.g., Ford-Fulkerson (by finding an augmenting path with O(E max| f |) time complexity) or Edmonds-Karp (by using bfs to find the shortest augmenting path, with O(VE2) time complexity) to solve the max-flow problem, typical implementations of these algorithms run pretty fast, in polynomial time in V, E.
